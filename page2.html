HTML CSS

INTRODUCTION TO CSS

CSS is the language we use to style an HTML document.
CSS describes how HTML elements should be displayed.

THE CSS UNIVERSAL SELECTOR

The universal selector (*) selects all HTML elements on the page.

THE CSS "BOX-SIZING" PROPERTY

The CSS box-sizing property allows us to include the padding and border in an element's total width and height.

If you set "box-sizing: border-box" on an element, padding and border are included in the width and height

THE CSS "MARGIN" PROPERTY

The CSS margin properties are used to create space around elements, outside of any defined borders.

HTML "Body element" set to : "margin: 0"

Purpose: Most browsers have default margins for the body (usually around 8px).
By setting margin: 0, you eliminate this default spacing, which is often needed for layout precision and to avoid unintended gaps.

HTML "Body element" set to : "color: black"

Purpose: Ensures that all text elements (such as paragraphs, headings, etc.) will have black text unless otherwise specified by other styles.

HTML "Body element" set to : "background-color: white"

Purpose: It ensures the webpage's background is white

HTML "NAV element" set to : "display: flex"

Purpose: Flexbox is used here to create a responsive layout for the navigation bar.

HTML "NAV element" set to : "flex-wrap: wrap"

Purpose: When the window is resized or viewed on smaller screens, the navigation items (e.g., links) will break into multiple lines instead of overflowing or shrinking too small to be readable. It improves responsiveness and accessibility.

HTML "NAV element" set to : "align-items: center"

Purpose: Ensures that the items in the navigation bar are centered vertically, so they appear evenly aligned and balanced regardless of their size or content.

HTML "NAV element" set to : "justify-content: center"

Purpose: By centering the items horizontally, it ensures that the navigation bar looks balanced, with the items aligned in the middle, whether there are only a few items or many.

HTML "NAV element" set to : "padding: 0.5rem"

Purpose: The padding adds space inside the "nav" element between the content (navigation items) and the edge of the navigation bar. This creates a buffer so that the items don't touch the edges, making the navigation bar visually cleaner.

HTML "NAV element" set to : "gap: 0.5rem"

Purpose: The gap creates space between the individual items in the navigation bar, making them easier to distinguish and more visually appealing.

HTML "NAV element" set to : "border-bottom: solid 1px #aaa"

Purpose: The bottom border visually separates the navigation bar from the rest of the content below it. It creates a subtle visual distinction, adding structure to the layout without being too bold.
